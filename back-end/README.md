#### Setup
- Node.JS 14 ( or later )
- Install dependencies `yarn` or `npm i`
- `npm start` - Runs project.
- `npm run start-watch` - Runs project(nodemon).
- `npm test` - Runs tests.
- `npm run test-watch` - Runs tests(nodemon).
- `npm run coverage` - Runs code coverage.
